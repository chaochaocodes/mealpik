# README

This README documents the steps necessary to get the
application up and running.

* Configuration
    Run 'bundle install' to install Gemfile
    Run 'rails db:migrate'
* Database creation
    Run 'rails db:seed'
* Database initialization
    Go to localhost:3000/landing
